JavaScript学习笔记

发表于2020-11-16|开发知识

输出

  1. alert()

弹框信息，内部可以执行语句。

  1. document.write()

将内容写到 HTML 文档里面。里面可以执行语句，也可以直接输出字符串，符合 HTML 规范可以解析语句。

  1. innerHTML

用来写入到HTML语句里面，可以先使用 document.getElementById(id) 来获得对应 ID 的 HTML 元素，然后使用 innerHTML 来获取这个元素的内容。也就是说，复合语句： document.getElementById(id).innerHTML 本身就相当与一个元素，可以输出它，也可以修改这个元素的值。

  1. console.log()

将里面的内容输出到浏览器的控制台中。

语法

  1. 使用 var 来定义一个变量。

  2. 使用 // 来进行单行注释，使用 /**/ 进行多行注释。

  3. 使用关键词 function 来定义一个函数。

  4. 单引号和双引号互通，都可以对字符串使用。

  5. 和 PHP 一样有强相等和弱相等。

事件

事件 描述
onchange HTML 元素改变
onclick 用户点击 HTML 元素
onmouseover 用户在一个HTML元素上移动鼠标
